## Installation

```bash
$ npm install
# or
$ yarn install
```

## Running the app

```bash
# development
$ npm run start
# or
yarn start

# watch mode
$ npm run start:dev
# or
yarn start:dev

# production mode
$ npm run start:prod
# or
yarn start:prod

# both client & server
$ npm run dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
